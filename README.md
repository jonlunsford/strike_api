# Strike API

API wrapper for the Strike Search website (http://getstrike.net)

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'strike_api'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install strike_api
